Comment participer ?

1. Être résident normand

Pour participer, vous devez résider en Normandie (départements 14, 27, 50, 61, 76).

2. Créer un compte sur chess.com

Si vous n'avez pas encore de compte, créez-en un sur chess.com pour pouvoir participer au tournoi.

3. S'inscrire au club

Remplissez le formulaire d'inscription pour rejoindre le club Normandie EEL.

4. Accepter l'invitation

Vous recevrez une invitation sur chess.com pour rejoindre le club. Acceptez-la pour finaliser votre inscription.

5. Rejoindre le tournoi

Une fois membre du club, rejoignez le tournoi via le lien du tournoi. Les inscriptions ouvrent 1h avant le début.
